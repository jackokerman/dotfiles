# Dotfiles

My personal dotfiles. 

## Installation

1. Clone this repository:

```bash
git clone https://github.com/jackokerman/dotfiles.git
cd dotfiles
```

2. Run the install script with desired options

```bash
# To create symlinks only
./install.sh link

# To set up shell tools (Zap, fzf, bat)
./install.sh shell

# To do both
./install.sh all
```
